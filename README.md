# 레트로 오크 던전

HTML, CSS, JavaScript, Canvas 2D만 사용하는 90년대 DOS풍 1인칭 던전 액션 프로토타입입니다.

## 실행

`index.html`을 브라우저에서 바로 열거나, 간단한 정적 서버로 폴더를 열면 됩니다.

조작:

- `WASD`: 이동
- `좌/우 방향키`: 시점 회전
- `마우스`: 캔버스 클릭 후 시점 회전
- `좌클릭 / Space`: 일반 검 공격
- `우클릭`: 분노 40을 소모해 특수공격
- `E`: 가까운 마을 NPC와 대화
- `Enter`: 시작 또는 게임오버 후 재시작

## 게임 내용

- 하나의 큰 던전 필드를 돌아다니며 사냥하고 성장합니다.
- 시작 구역은 마을 역할을 하는 안전지대이며, NPC와 간단히 대화할 수 있습니다.
- 몬스터는 오크, 스켈레톤, 오우거, 워록, 오크 대장, 스켈레톤 킹이 등장합니다.
- 스켈레톤은 빠르고 약하며, 오우거는 느리지만 튼튼하고 강합니다.
- 워록은 거리를 두고 마법탄을 발사합니다.
- 몬스터를 처치하면 경험치를 얻고 레벨업합니다.
- 레벨업하면 최대 체력, 분노 최대치, 공격력이 성장합니다.
- 분노는 특수공격 한 번에 전부 사라지지 않고, 특수공격마다 40씩 소모됩니다.
- 처치된 몬스터는 일정 시간이 지나면 플레이어와 떨어진 위치에서 다시 등장합니다.
- 일반 몬스터와 보스급 몬스터는 체력 회복 아이템을 떨어뜨릴 수 있습니다.
- 몬스터는 체력 물약, 분노 결정, 경험치 구슬, 강화 주문서를 떨어뜨릴 수 있습니다.
- 강화 주문서를 먹으면 즉시 검이 `+1` 강화되고, 색과 크기와 공격력이 올라갑니다.
- 죽으면 마을에서 부활하며, 강화된 검은 죽은 자리에 떨어집니다.
- 떨어진 검을 다시 찾아가면 기존 강화 수치를 회수할 수 있습니다.
- 검은 레벨에 따라 흰검, 붉은검, 황금검으로 강화됩니다.
- 미니맵은 벽, 플레이어, 몬스터, 아이템, 투사체를 표시합니다.

## GitHub Pages 배포

정적 사이트이므로 GitHub Pages로 바로 배포할 수 있습니다.

1. GitHub 저장소를 엽니다.
2. `Settings`로 이동합니다.
3. `Pages` 메뉴를 엽니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. `main` 브랜치와 `/` 루트 폴더를 선택합니다.
6. 저장하면 Pages 주소가 생성됩니다.
