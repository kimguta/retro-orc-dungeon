# Paper Citadel

한국어명은 `종이성채`입니다.

링크를 열고 성채를 달리며 몬스터를 사냥하고, 죽어도 세이프티 존에서 다시 출발해 발록을 반복 트라이하는 웹 기반 레트로 던전 액션 프로토타입입니다.

현재 버전은 HTML, CSS, JavaScript, Canvas 2D만 사용하는 싱글 플레이 기반입니다. 이후 캐릭터명 저장과 친구용 멀티 레이드 구조로 확장할 예정입니다.

## 실행

`index.html`을 브라우저에서 바로 열거나 정적 서버로 폴더를 열면 됩니다.

조작:

- `WASD`: 이동
- `좌/우 방향키`: 시점 회전
- `마우스`: 캔버스 클릭 후 시점 회전
- `좌클릭 / Space`: 일반 공격
- `우클릭`: 분노를 소모한 특수공격
- `E`: 세이프티 존 NPC와 대화하고 HP 회복
- `Enter`: 시작 또는 재시작

## 현재 방향

- 게임명은 `Paper Citadel` / `종이성채`입니다.
- 캐릭터와 몬스터는 이미지 파일 없이 Canvas에서 직접 그립니다.
- 기존 몬스터의 귀여운 실루엣을 유지하면서 종이 판넬과 보드게임 말 같은 느낌을 강화합니다.
- 검은 골판지 검처럼 보이도록 다듬고, 강화 수치에 따라 색이 계속 변합니다.
- 세이프티 존에서 출발해 성채 깊은 곳의 발록을 향해 전진합니다.
- 안전지대 가까운 몬스터는 약하고 발록 쪽으로 갈수록 강한 몬스터와 중간보스가 많아집니다.
- 몬스터와 보스는 시간이 지나면 다시 등장합니다.
- 죽어도 검 강화는 유지되고, 세이프티 존에서 다시 발록 트라이를 이어갑니다.
- 회복 물약은 최대 HP 비율로 회복되어 고레벨에서도 의미가 있습니다.

## 성장

- 플레이어 최대 레벨은 `99`입니다.
- 검 강화 최대 수치는 `+200`입니다.
- 레벨업 시 최대 HP, 공격력, 분노 수치가 성장하고 HP가 회복됩니다.
- 검 색상은 기본 철검에서 붉은색, 황금색, 초록색, 푸른색, 보라색을 거쳐 `+200`에서 검은 최종 강화 검으로 변합니다.

## GitHub Pages 배포

정적 사이트이므로 GitHub Pages로 배포할 수 있습니다.

1. GitHub 저장소를 엽니다.
2. `Settings`로 이동합니다.
3. `Pages` 메뉴를 엽니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. `main` 브랜치와 `/` 루트 폴더를 선택합니다.
6. 저장하면 Pages 주소가 생성됩니다.
